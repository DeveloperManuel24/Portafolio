Proyecto Manejo Presupuesto

Tecnologías Usadas

.NET, ASP.NET Core, AutoMapper, Dapper, Identity, Razor Pages, SQL Server, ClosedXML

Descripción del Proyecto

El proyecto "Manejo de Presupuesto" es una aplicación web para gestionar finanzas personales. Utiliza .NET y ASP.NET Core en el backend, con AutoMapper para el mapeo de objetos y Dapper para el acceso a datos. La aplicación permite a los usuarios registrar y clasificar transacciones financieras, generar reportes detallados por día, semana y mes, y exportar estos reportes a archivos Excel. También ofrece funcionalidades de autenticación y autorización mediante Identity, y utiliza interfaces de repositorios para la gestión de cuentas, tipos de cuentas y categorías. La interfaz de usuario se implementa con Razor Pages y controles de formularios ASP.NET, proporcionando una experiencia de usuario intuitiva para la gestión financiera.
