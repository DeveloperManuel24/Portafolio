Proyecto Tienda Virtual

Tecnologías Usadas

Strapi, Remix.js, PostgreSQL (con una base de datos en Render), React, Cloudinary, styled-components

Descripción del Proyecto

El proyecto GuitarLA es una plataforma de comercio electrónico para la venta de guitarras, gestionada con Strapi para la administración de contenido en el backend y Remix.js para la renderización del frontend. Utiliza PostgreSQL como base de datos, alojada en Render.com. La aplicación ofrece listados de productos, un blog, cursos y un carrito de compras, con manejo robusto de errores y etiquetas meta optimizadas para SEO.
