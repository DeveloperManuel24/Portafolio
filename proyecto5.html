Proyecto Quiosco de Comida

Tecnologías Usadas

Next.js, Tailwind CSS, postgres(Con una base de datos en Render), Prisma, react-modal, react-toastify, swr

Descripción del Proyecto

Este proyecto consiste en una aplicación web desarrollada con Next.js y Tailwind CSS para el frontend, y Postgres gestionado por Prisma para el backend. La aplicación permite a los usuarios gestionar productos, categorías y órdenes en un quiosco. Además, se utilizan react-modal para la creación de modales, react-toastify para notificaciones, y swr para la gestión de datos con caching y revalidación.
