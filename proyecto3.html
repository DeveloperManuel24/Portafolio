Aplicacion de Registro de Noticias

Tecnologías Usadas

.NET Blazor, Dapper, ClosedXML, Npgsql, PostgreSQL, ASP.NET Core MVC, JavaScript,Leaflet, HTML, CSS, Bootstrap, Microsoft.Extensions.Configuration, System.ComponentModel.DataAnnotations

Descripción del Proyecto

El proyecto es una aplicación web de gestión de noticias periodísticas desarrollada con .NET Blazor. Esta aplicación permite a los usuarios ingresar, almacenar y gestionar diversas noticias periodísticas, proporcionando una plataforma organizada y eficiente para la gestión de contenido. El sistema cuenta con una funcionalidad avanzada que permite la creación y edición de noticias, incluyendo detalles como el título, la URL de la fuente, el nombre de la fuente, fechas de publicación y de los hechos, así como la ubicación geográfica y otros datos relevantes. Además, las noticias pueden ser categorizadas mediante palabras clave, cada una de las cuales está asociada a un icono representativo almacenado en una estructura de carpetas dentro del proyecto. La aplicación también incluye una robusta funcionalidad de filtrado y exportación, permitiendo a los usuarios generar informes detallados en formato Excel basados en filtros específicos como el título de la noticia, la palabra clave, las fechas y el país. Esta característica es crucial para el análisis y la presentación de datos periodísticos de manera estructurada. El proyecto emplea tecnologías modernas como Dapper para el acceso a datos y ClosedXML para la generación de archivos Excel, asegurando eficiencia y alta performance en el manejo de grandes volúmenes de datos. Con una interfaz de usuario intuitiva y amigable, la aplicación está diseñada para facilitar la gestión de contenido periodístico, permitiendo a los periodistas y gestores de contenido mantener un control preciso sobre la información y mejorar la visibilidad y accesibilidad de las noticias.
